# GiggleWater

## Description
An application for a 20s themed vodka and gin bar that allows a user to add, edit, and delete types of vodka and gin. The application also allows a user to see all of the choices on offer and sort by preference.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 1.0.0.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
